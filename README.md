# POC - git5-Android-Java-RS-Payload-Calculator
Aplicación Java de consola que combina una calculadora funcional con un comportamiento de red en segundo plano como **prueba de concepto académica**.
El proyecto explora **intrusión, control y evasión por contexto**, demostrando cómo software trivial puede ocultar lógica crítica.
Diseñado para análisis de código, threat modeling y concientización en entornos educativos autorizados.
